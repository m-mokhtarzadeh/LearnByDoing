# Create API key to configure AWS CLI access
To Create access key and secret key using following instruction.
## 1. Select IAM Service from servies menu
![Select IAM Service](./img/CreateAPIKey/1-SelectIAMServie.png)
## 2. Select Users Section from navigation menu
![Select User Menu](./img/CreateAPIKey/2-GoToUserSettings.png)
## 3. Select proper IAM user to generate API Key
![Select Proper User](./img/CreateAPIKey/3-SelectProperUser.png)
## 4. Choose User Create Access Key
![Generate Access Key ](./img/CreateAPIKey/4-CreateAccessKey.png) 
## 5. Select Access Key for Using CLI Access
![CLI Access](./img/CreateAPIKey/5-ChooseCLIAccess.png)
## 6. Set Description for Genrated CLI Key
![Select Description](./img/CreateAPIKey/6-SetDescription.png)
## 7. Show secret key and download it as csv file
![Download Secret Key](./img/CreateAPIKey/7-ShowAndDownloadKey.png)
